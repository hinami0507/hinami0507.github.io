匀速动画

animate1-长度运动
animate2-不透明度运动

一个动作startMove函数的基本流程：

  • clearInterval(timer) 先清理计时器，避免多次重复触发动作
  • 设置计时器
  • 计时器内部获取目标长度和当前长度，进行比对，相同则清理计时器，停止动画，不同则长度递增

变速缓冲运动

animate3-变速缓冲运动

在animate1的基础上改进在于 速度值来源于 目标值-当前值，
并且要对速度值取整，整的向上取整，负的向下取整。

IE9, Firefox, Chrome, Opera 和 Safari 使用属性 opacity 来设定透明度。opacity 属性能够设置的值从 0.0 到 1.0。值越小，越透明。
IE8 以及更早的版本使用滤镜 filter:alpha(opacity=x)。x 能够取的值从 0 到 100。值越小，越透明。

多物体运动

多物体运动，不能共用计时器
animate4-多物体运动

可以发现，把鼠标移动到方块1上，然后迅速，移动到方块2上，发现方块1的动画也停止了，因为他们共用计时器导致的。

所以要为每一个动画开一个计时器

animate4-fix1

这样每一个动画都有自己的计时器，不会冲突。

然后将不透明度和长度变化的动画进行融合：
在startMove中对传入的attr参数进行判断，然后做出相应的改变

1、获取当前透明度不用parseInt
2、设置透明度要考虑兼容
obj.style.filter=’alpha(opacity:’+(当前透明度+变化速度)+’)’;
obj.style.opacity=(当前透明度+变化速度)/100;
3、透明度不加“px”
在使用parseInt()时处理透明度小数时,会有影响 单位设置
相应位置进行判断 IE/FireFox
取相应值 Math.round()四舍五入取整数值
Math.round(parseFloat(getStyle(obj,attr))*100)

animate6-链式运动

在原来基础上多传入一个回调函数fn，在动画停止的时候执行

function startMove(obj, json,workspeed,fn) {
clearInterval(obj.timer);

obj.timer = setInterval(function() {
    var flag = true;
    for (var attr in json) {

        if (attr == 'opacity') {
            var icur = Math.round(parseFloat(getstyle(obj, attr)) * 100);
        } else {
            var icur = parseInt(getstyle(obj, attr));
        }
        var speed = (json[attr] - icur) / 8;
        speed = speed > 0 ? Math.ceil(speed) : Math.floor(speed);

        if (json[attr] != icur) {
            flag = false;
        }
        if (flag)
        {
            clearInterval(obj.timer);
            if(fn) fn();
        }
        else {
            if (attr == 'opacity') {
                obj.style.filter = 'alpha(opacity:' + icur + speed + ')';
                obj.style.opacity = (icur + speed) / 100;
            } else
                obj.style[attr] = icur + speed + 'px';
        }
    }

}, workspeed)

}

function getstyle(obj, attr) {
if (obj.currentStyle)
return obj.currentStyle[attr];
else
return getComputedStyle(obj, false)[attr];
}
